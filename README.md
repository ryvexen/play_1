# Vintage Book Effect with Turn.js

This project implements a vintage-style book effect using Turn.js, featuring:

## Features
- Authentic vintage book design with aged paper texture
- Realistic page-turning animation
- Ornate book covers with decorative elements
- Page headers, numbers, and chapter divisions
- Keyboard navigation support (arrow keys)
- Previous/Next buttons for navigation

## Design Elements
- Aged paper background with subtle texture
- Vintage color palette (browns, golds, and sepia tones)
- Decorative ornaments and styling
- Classic typography with Times New Roman and Georgia fonts
- Simulated parchment texture

## How to Use
1. Open your browser and navigate to `http://localhost:8000`
2. Click on the right edge of a page to turn it forward
3. Click on the left edge of a page to turn it backward
4. Use the Previous/Next buttons below the book
5. Use left/right arrow keys on your keyboard

## Technical Implementation
- Uses Turn.js library for realistic 3D page flipping
- Responsive design that works on different screen sizes
- Pure HTML, CSS, and JavaScript implementation
- No external dependencies beyond jQuery and Turn.js
